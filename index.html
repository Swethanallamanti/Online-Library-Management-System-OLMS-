Online Library Management Systeem is an Automated Library System that handles the various
functions of the Library. It provides a complete solution to the library management software.

Rules you should know!

  1. Searh for the book before you proceed to apply.
  2. Book can be applied only it is present in the library.
  3. Request can be accepted or rejected only by the admin.
  4. When Book Request accepted by the admin student must go to the library and collect the book.
  5. If in case book Request rejected by the admin wait for some days and Re-apply for the book.
  6. Student must submit the book before due date. If incase not submitted fee will be collect.
